#### 基础标签
##### 输入与选择标签
```html
<input type="输入类型">
```
|属性补全|作用|备注属性|
|-------|-----|------|
|type="text"|单行文本输入||
|type="password"|密码输入||
|type="email"|邮件输入||
|type="number"|数字输入|min="最小值" max="最大值"|
|type="date"|日期选择||
|type="checkbox"|复选||
|type="radio"|单选||
|type="submit"|提交||
|type="reset"|重置||
|type="button"|按钮||
|type="file"|文件上传||
|type="hidden"|隐藏输入||
|type="color"|颜色选择器||
|type="range"|滑动条|min="最小值" max="最大值"|
|type="search"|搜索框||
|type="tel"|电话输入||
|type="url"|网址输入||
